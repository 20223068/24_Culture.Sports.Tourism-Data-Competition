# 2024년 문화체육관광 데이터 활용대회
문화예술활동 지역 격차 분석 프로젝트

---

## 프로젝트 배경

코로나19 이후 문화예술 활동은 회복되고 있으나,  
지역 간 회복 속도 및 참여 수준의 격차가 심화되고 있습니다.

- 수도권 중심 문화 집중
  - 공연 62%, 매출 86% 집중
- 비수도권(특히 전라도)
  - 문화시설 대비 이용률 저조
  - 문화 소비 금액(VLM) 낮음
  - 교통 불편 → 주요 참여 장애 요인

**핵심 문제**  
> 문화 격차는 시설 부족이 아니라, 활용과 접근성의 문제

---

## Data

### 1. 카드 소비 데이터
- 신한카드 내국인 데이터 (2023.07 ~ 2024.09)
- 문화예술활동 관련 소비 금액(VLM)

### 2. 문화활동 데이터
- 국민문화예술활동 조사 (2023)
- 관람 여부, 이동성, 참여 장애 요인

### 3. 문화시설 데이터
- 전국 문화기반시설 총람 (2023)
- 시설 종류 및 연간 이용자 수

### 4. GIS 데이터
- 지역 코드 기반 지도 매핑

---

## Methodology

### 1. Data Preprocessing
- 지역 코드 매핑 및 정규화
- 문화 관련 소비 데이터 필터링
- 문화시설 데이터 통합 (9종 시설)
- 교통 관련 변수 추출

### 2. Feature Engineering
- 문화 소비 수준 (VLM)
- 문화시설 밀도
- 연간 이용자 수
- 교통 불편 비율
- 지역 간 이동성

### 3. Clustering (K-Means)
지역별 문화활동 특성을 기반으로 군집화 수행

#### Cluster result

| Cluster | 특징 |
|--------|------|
| Cluster 1 | 시설 있음 / 소비 낮음 / 이용 낮음 |
| Cluster 2 | 시설 부족 / 소비 낮음 / 이용 낮음 |
| Cluster 3 | 문화활동 활성화 지역 |
| Cluster 4 | 문화활동 활성화 지역 |

---

## Key Findings

### 1. 전라도 지역 특성
- 시설 대비 문화 소비 수준 낮음
- 문화 참여로 이어지지 않는 구조

### 2. 교통의 영향
- 교통 불편 지역일수록 타 지역 이동 감소
- 문화 활동 참여율 감소

### 3. 핵심 Insight
> 문화 격차의 본질은 인프라 부족이 아니라  
> **접근성과 참여 행동의 문제**

---

## Proposed Solutions

### Solution 1: 찾아가는 문화 서비스 (Cluster 2)
- 대상: 문화 접근성이 낮은 지역
- 방식:
  - 순회 공연 / 전시 / 체험 프로그램 제공
  - 군집 기반 우선순위 선정

**기대 효과**  
- 문화 경험 기회 확대
- 문화 관심도 증가

### Solution 2: 문화교통카드 (Cluster 1)
- 대상: 시설은 있으나 참여도가 낮은 지역
- 방식:
  - 문화활동 인증 시 교통비 페이백
  - 활성 지역(Cluster 3, 4) 이동 유도

**기대 효과**
- 문화 활동 참여 증가
- 지역 간 문화 이동 활성화

---

## Expected Impact
- 지역 간 문화 격차 완화
- 교통 장벽 감소
- 문화 참여율 증가
- 균형 있는 지역 발전
